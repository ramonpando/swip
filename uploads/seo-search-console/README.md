# Exportaciones de Google Search Console

Sube los archivos CSV sin cambiarles el nombre y separados por dominio:

- `swip/`: exportación completa de la propiedad de SWIP.
- `kevant/`: exportación completa de la propiedad de Kevant.

Cada exportación puede incluir estos archivos:

- `Aparición en búsquedas.csv`
- `Consultas.csv`
- `Dispositivos.csv`
- `Filtros.csv`
- `Gráfico.csv`
- `Páginas.csv`
- `Países.csv`

Usaremos todo el historial disponible aunque el dominio todavía no tenga 16 meses de datos.
